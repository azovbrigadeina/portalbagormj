Pengaturan Database Portal

Anda bisa menyinkronkan portal ini dengan Google Spreadsheet milik Anda untuk mengelola menu link secara dinamis tanpa mengubah kode.

ID terletak di URL Google Sheet Anda: https://docs.google.com/spreadsheets/d/[SPREADSHEET_ID]/edit

Cara Penyiapan Google Sheets:

  1. Salin template Google Sheet resmi untuk aplikasi portal ini.
  2. Klik tombol bagikan (**Share**) di Google Sheet Anda, set akses menjadi "Anyone with the link can view" (Siapa saja yang memiliki link dapat melihat).
  3. Salin ID Spreadsheet dari URL-nya dan tempel di atas.
  4. Klik tombol **Simpan & Sinkronkan**.
